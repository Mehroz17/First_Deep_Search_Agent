## ğŸš€ Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Mehroz17/First_Deep_Search_Agent.git
cd Deep-Search-Orchestration-Agent
```
### 2. Initialize UV 
```
uv init
```
### 3. Activate UV Venv
```
source .venv/bin/activate   
```
### 4. Add all the libraries
```
uv sync
```
### 5. Run Main File
```
uv run python main.py
```
